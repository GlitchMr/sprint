sprint by GlitchMr

sprintf() implementation for JavaScript

Have you ever programmed in C++? You probably have, but if you didn't... have you programmed in PHP? The answer is most likely yes. You have probably seen sprintf() function. You may have tried using it in JavaScript. 

>>> sprintf('%s', 'hello');
    Unhandled Error: Undefined variable: sprintf

OK, perhaps not. Well, now you can use sprint() as replacement for sprintf() thanks to this project.

The project itself uses CoffeeScript, but because CoffeeScript compiles to JavaScript, you can use this project without having CoffeeScript compiler installed.

Any issues? Something not working like in C++/PHP/Perl/Ruby/whatever? Report it to issue tracker and it will be hopefully fixed in next release.

Install

On website:

  1. Upload lib/sprint.js somewhere on your server (2KB after GZIPing).
  2. <script src="lib/script.js"></script>

In Node.js:

  1. npm install sprint in console (requires npm)
  2. var sprint = require('sprint').sprint;
